# Blinkit Sales Analysis

## 📊 Project Overview
This project analyzes Blinkit sales data to identify key business insights using Power BI.  
The dashboard helps understand sales performance, product demand, and customer trends.

## 🔧 Tools Used
- Power BI
- Excel / CSV
- Data Visualization

## 📁 Dataset
The dataset includes:
- Order details
- Product categories
- Sales amount
- Quantity sold
- City / Region
- Date information

## 📈 Dashboard Features
- Total sales and orders overview
- Category-wise sales analysis
- City-wise performance
- Monthly and yearly trends
- Interactive filters and slicers
